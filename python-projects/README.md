# Security-Focused Python Projects

Small Python scripts and study exercises used to practise cybersecurity concepts, automation, and defensive thinking.

## Projects

- `Password Audit Tool.py` - Checks password strength using common security principles.
- `password-generator.py` - Generates random passwords with configurable complexity.
- `Port Scanner..py` - Basic TCP port scanner for networking practice.
- `Log file.py` - Log parsing practice for extracting useful security information.
- `SOC Analyst Simulator.py` - Small simulation of SOC alert review tasks.
- `LoginFirewall_exc.py` - Login and access-control themed exercise.
- `malware_study.py` - Study notes and prompts around malware concepts.
- `physical_security_study.py` - Security+ physical security revision.
- `security_fundamentals_study.py` - Security fundamentals revision.
- `social_engineering_study.py` - Social engineering revision.
- `threat_actors_study.py` - Threat actor revision.

General programming exercises have been moved to the `bsc-cybersecurity` coursework repository so this repo stays focused on security labs and Blue Team practice.
